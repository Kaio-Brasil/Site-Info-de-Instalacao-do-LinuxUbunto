O QUE É SOFTWARE LIVRE

Software livre é qualquer programa que pode ser executado, copiado, estudado, modificado e redistribuído gratuitamente e ainda ter a possibilidade de acessar ao seu código fonte livremente, podendo ser alterado conforme as suas necessidades. Essa é a definição genérica do software livre. Para ser um software livre o programa ou aplicativo deve respeitar as quatro liberdades essenciais definidas pela Free Software Foundation, que em português quer dizer Fundação para Software Livre, uma organização sem fins lucrativos que se dedica a eliminar as restrições sobre cópias, estudos e modificações dos programas. São pré-requisitos para serem software livre:

  • A liberdade de executar o programa como quiser.
  • A liberdade de estudar programas e conhecer seu código-fonte.
  • A liberdade de redistribuir copias.
  • A liberdade de distribuir copias de seus projetos.

É necessário que o programa ou aplicativo seja distribuído com o seu código fonte para cumprir as quatro liberdades de um software livre e que não tenha restrições para que seja alterado e redistribuído esse código. A maneira comum de distribuição de software livre é anexar uma licença de software livre. Porque tais diretrizes são centrais ao conceito, o qual se opõe ao conceito de software proprietário. Muitas gentes ainda confundem software livre como software gratuito, pois alguns softwares livres são disponibilizados gratuitamente, mas isso não impede que ele seja comercial. A maior vantagem do software livre é o compartilhamento de conhecimento, muitos desenvolvedores se beneficiam disso e a comunidade de desenvolvedor só aumenta.

Exemplos de software livre: Mozilla Firefox, GIMP, Ubuntu.

COPYLEFT

É um tipo de licença onde o autor cede alguns direitos, ou seja, continua sendo dono, mas respeitando o conceito das quatro liberdades encontradas na Free Software Foundation. Isso significa que o usuário tem a permissão para possuir uma cópia de uma obra dando a liberdade para copiar, modificar e redistribuir. O que o copyleft diz é que qualquer um que distribui o software, com ou sem modificações, tem que passar adiante, respeitando a filosofia do software livre, garantindo a liberdade para todos.

O QUE É LINUX

É normal achar que o Linux é um sistema operacional, e essa definição não está errada, mas o Linux é um núcleo do sistema chamamos de Kernel, a parte principal do sistema operativo do computador, que tem a função de conectar o software ao hardware e estabelecer uma comunicação entre os recursos do sistema. Foi inspirado no sistema minix tem seu código fonte aberto, estando disponível sob licença GPL, ou seja, qualquer pessoa pode utilizar, estudar, modificar e distribuir livremente. O Linux tem várias distribuições, essas distribuições são um conjunto de programas mais o Kernel, em outras palavras, as distribuições são os sistemas operacionais do Linux. Atualmente o Linux tem vários colaboradores, a Canonical é um desses colaboradores e responsável pela distribuição do Ubuntu Linux disponível gratuitamente em seu site. Usaremos uma cópia do Ubuntu para instalar o sistema Linux. Conhecido como Ubuntu Linux, o Ubuntu é um sistema operacional de código aberto construído a partir do núcleo Linux, é baseado em Debian e segue as normas de software livre. Lançado em 2004, sua proposta é oferecer um sistema que possa ser utilizado sem dificuldades, independentemente de nacionalidade, nível de conhecimento ou limitações físicas. Um sistema seguro, que contém vários aplicativos, como navegador, editor de texto, planilha eletrônica, apresentação de slide, e entre outros. Atualmente está na versão 16.04LTS (Long Term Support) e seu suporte é de 5 anos para desktop e servidores, as novas versões do Ubuntu são lançadas periodicamente duas vezes ao ano, uma no mês de abril e outra no mês de outubro. Por ser um sistema livre, o Ubuntu é considerado uma das distribuições Linux mais usadas no mundo.
